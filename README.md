# 使い方

<img src="https://github.com/akita11/MinimumContChecker/blob/main/battery.jpg" width="240px">

LR41ボタン電池を2個、このように入れます（向きに注意してください）。

上部の電源スイッチを押すと緑LEDが点灯します。

<img src="https://github.com/akita11/MinimumContChecker/blob/main/cont.jpg" width="240px">

2本のテスターリードをチェック箇所にあて、2点が短絡（およそ1Ω以下）していると、ブザーが鳴り、黄LEDが点灯します。

使用後は電源スイッチをもう一度押してオフにしておいてください。
